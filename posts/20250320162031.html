基本积分公式

1. 幂函数积分

∫xndx=xn+1n+1+C(n≠−1)\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1) ∫xndx=n+1xn+1​+C(n=−1)

∫1xdx=ln⁡∣x∣+C\int \frac{1}{x} dx = \ln|x| + C ∫x1​dx=ln∣x∣+C

2. 指数函数积分

∫exdx=ex+C\int e^x dx = e^x + C ∫exdx=ex+C

∫axdx=axln⁡a+C(a>0,a≠1)\int a^x dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \neq 1) ∫axdx=lnaax​+C(a>0,a=1)

3. 三角函数积分

∫sin⁡xdx=−cos⁡x+C\int \sin x dx = -\cos x + C ∫sinxdx=−cosx+C

∫cos⁡xdx=sin⁡x+C\int \cos x dx = \sin x + C ∫cosxdx=sinx+C

∫sec⁡xdx=ln⁡∣sec⁡x+tan⁡x∣+C\int \sec x dx= \ln|\sec x+ \tan x|+C ∫secxdx=ln∣secx+tanx∣+C

∫csc⁡xdx=ln⁡∣csc⁡x+cot⁡x∣+C\int \csc x dx= \ln|\csc x+ \cot x|+C ∫cscxdx=ln∣cscx+cotx∣+C

∫tan⁡xdx=−ln⁡∣cos⁡x∣+C\int \tan x dx = -\ln|\cos x| + C ∫tanxdx=−ln∣cosx∣+C

∫cot⁡xdx=ln⁡∣sin⁡x∣+C\int \cot x dx = \ln|\sin x| + C ∫cotxdx=ln∣sinx∣+C

∫1cos⁡2xdx=∫sec⁡2xdx=tan⁡x+C\int \frac{1}{\cos^2 x} dx=\int \sec^2 x dx = \tan x + C ∫cos2x1​dx=∫sec2xdx=tanx+C

∫1sin⁡2x=∫csc⁡2xdx=−cot⁡x+C\int \frac{1}{\sin^2 x} =\int \csc^2 x dx = -\cot x + C ∫sin2x1​=∫csc2xdx=−cotx+C

∫sec⁡xtan⁡xdx=sec⁡x+C\int \sec x \tan x dx = \sec x + C ∫secxtanxdx=secx+C

∫csc⁡xcot⁡xdx=−csc⁡x+C\int \csc x \cot x dx = -\csc x + C ∫cscxcotxdx=−cscx+C

4. 对数函数积分

∫ln⁡xdx=xln⁡x−x+C\int \ln x dx = x \ln x - x + C ∫lnxdx=xlnx−x+C

5. 反三角函数积分

∫11−x2dx=arcsin⁡x+C\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C ∫1−x2​1​dx=arcsinx+C

∫11+x2dx=arctan⁡x+C\int \frac{1}{1+x^2} dx = \arctan x + C ∫1+x21​dx=arctanx+C

6. 其他常见积分

以下积分都是通过换元推出来的，所以很难记，能记还是记一下，记不住就现场推

∫1x2+a2dx=1aarctan⁡(xa)+C\int \frac{1}{x^2 + a^2} dx = \frac{1}{a} \arctan\left(\frac{x}{a}\right) + C ∫x2+a21​dx=a1​arctan(ax​)+C

这个不用记，令x=atan⁡xx=a\tan xx=atanx,可以直接得到结果

∫1x2−a2dx=12aln⁡∣x−ax+a∣+C\int \frac{1}{x^2 - a^2} dx = \frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C ∫x2−a21​dx=2a1​ln​x+ax−a​​+C

∫1a2−x2dx=12aln⁡∣x+ax−a∣+C\int \frac{1}{a^2-x^2} dx = \frac{1}{2a} \ln\left|\frac{x+a}{x-a}\right| + C ∫a2−x21​dx=2a1​ln​x−ax+a​​+C

这俩也不用记，通过裂项可以很容易得出结果

∫1x2+a2dx=ln⁡(x+x2+a2)+C\int \frac{1}{\sqrt{x^2 + a^2}} dx=\ln(x+\sqrt{x^2+a^2})+C ∫x2+a2​1​dx=ln(x+x2+a2​)+C

∫1x2−a2dx=ln⁡∣x+x2−a2∣+C\int \frac{1}{\sqrt{x^2 - a^2}} dx = \ln|x + \sqrt{x^2 - a^2}| + C ∫x2−a2​1​dx=ln∣x+x2−a2​∣+C

这俩也不用记，第一个令x=atan⁡tx=a\tan tx=atant,然后利用∫sec⁡xdx=ln⁡∣sec⁡x+tan⁡x∣+C\int \sec x dx= \ln|\sec x+ \tan x|+C∫secxdx=ln∣secx+tanx∣+C可以得到结果；第二个令x=asin⁡xx=a \sin xx=asinx,然后利用∫csc⁡xdx=ln⁡∣csc⁡x+cot⁡x∣+C\int \csc x dx= \ln|\csc x+ \cot x|+C∫cscxdx=ln∣cscx+cotx∣+C可以得到结果

∫1a2−x2dx=arcsin⁡(xa)+C\int \frac{1}{\sqrt{a^2-x^2}}dx=\arcsin(\frac{x}{a})+C ∫a2−x2​1​dx=arcsin(ax​)+C

这个更不用记了，令x=asin⁡xx=a \sin xx=asinx，连公式都不用直接得出来结果了
