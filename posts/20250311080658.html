一、基本初等函数的导数公式

  1. 常数函数
    f(x)=C⇒f′(x)=0f(x) = C \quad \Rightarrow \quad f'(x) = 0f(x)=C⇒f′(x)=0
  2. 幂函数
    f(x)=xn⇒f′(x)=nxn−1f(x) = x^n \quad \Rightarrow \quad f'(x) = n x^{n-1}f(x)=xn⇒f′(x)=nxn−1
  3. 指数函数
    • f(x)=ex⇒f′(x)=exf(x) = e^x \quad \Rightarrow \quad f'(x) = e^xf(x)=ex⇒f′(x)=ex
    • f(x)=ax⇒f′(x)=axln⁡a(a>0,a≠1)f(x) = a^x \quad \Rightarrow \quad f'(x) = a^x \ln a \quad (a > 0, a \neq 1)f(x)=ax⇒f′(x)=axlna(a>0,a=1)
  4. 对数函数
    • f(x)=ln⁡x⇒f′(x)=1xf(x) = \ln x \quad \Rightarrow \quad f'(x) = \frac{1}{x}f(x)=lnx⇒f′(x)=x1​
    • f(x)=log⁡ax⇒f′(x)=1xln⁡a(a>0,a≠1)f(x) = \log_a x \quad \Rightarrow \quad f'(x) = \frac{1}{x \ln a} \quad (a > 0, a \neq 1)f(x)=loga​x⇒f′(x)=xlna1​(a>0,a=1)
  5. 三角函数
    • f(x)=sin⁡x⇒f′(x)=cos⁡xf(x) = \sin x \quad \Rightarrow \quad f'(x) = \cos xf(x)=sinx⇒f′(x)=cosx
    • f(x)=cos⁡x⇒f′(x)=−sin⁡xf(x) = \cos x \quad \Rightarrow \quad f'(x) = -\sin xf(x)=cosx⇒f′(x)=−sinx
    • f(x)=tan⁡x⇒f′(x)=sec⁡2x=1cos⁡2f(x) = \tan x \quad \Rightarrow \quad f'(x) = \sec^2 x =\frac{1}{\cos^2}f(x)=tanx⇒f′(x)=sec2x=cos21​
    • f(x)=cot⁡x⇒f′(x)=−csc⁡2x=−1sin⁡2xf(x) = \cot x \quad \Rightarrow \quad f'(x) = -\csc^2 x= -\frac{1}{\sin^2 x}f(x)=cotx⇒f′(x)=−csc2x=−sin2x1​
    • f(x)=sec⁡x⇒f′(x)=sec⁡xtan⁡xf(x) = \sec x \quad \Rightarrow \quad f'(x) = \sec x \tan xf(x)=secx⇒f′(x)=secxtanx
    • f(x)=csc⁡x⇒f′(x)=−csc⁡xcot⁡xf(x) = \csc x \quad \Rightarrow \quad f'(x) = -\csc x \cot xf(x)=cscx⇒f′(x)=−cscxcotx
  6. 反三角函数
    • f(x)=arcsin⁡x⇒f′(x)=11−x2,x∈(−1,1)f(x) = \arcsin x \quad \Rightarrow \quad f'(x) = \frac{1}{\sqrt{1-x^2}}, \quad x \in (-1, 1)f(x)=arcsinx⇒f′(x)=1−x2​1​,x∈(−1,1)
    • f(x)=arccos⁡x⇒f′(x)=−11−x2,x∈(−1,1)f(x) = \arccos x \quad \Rightarrow \quad f'(x) = -\frac{1}{\sqrt{1-x^2}}, \quad x \in (-1, 1)f(x)=arccosx⇒f′(x)=−1−x2​1​,x∈(−1,1)
    • f(x)=arctan⁡x⇒f′(x)=11+x2,x∈(−∞,+∞)f(x) = \arctan x \quad \Rightarrow \quad f'(x) = \frac{1}{1+x^2}, \quad x \in \left( -\infty, +\infty \right)f(x)=arctanx⇒f′(x)=1+x21​,x∈(−∞,+∞)
    • f(x)=arccot⁡x⇒f′(x)=−11+x2,x∈(−∞,+∞)f(x) = \operatorname{arccot} x \quad \Rightarrow \quad f'(x) = -\frac{1}{1+x^2}, \quad x \in \left( -\infty, +\infty \right)f(x)=arccotx⇒f′(x)=−1+x21​,x∈(−∞,+∞)

二、导数的四则运算法则

  1. 加法法则
    (f(x)+g(x))′=f′(x)+g′(x)(f(x) + g(x))' = f'(x) + g'(x)(f(x)+g(x))′=f′(x)+g′(x)
  2. 减法法则
    (f(x)−g(x))′=f′(x)−g′(x)(f(x) - g(x))' = f'(x) - g'(x)(f(x)−g(x))′=f′(x)−g′(x)
  3. 乘法法则
    (f(x)⋅g(x))′=f′(x)g(x)+f(x)g′(x)(f(x) \cdot g(x))' = f'(x) g(x) + f(x) g'(x)(f(x)⋅g(x))′=f′(x)g(x)+f(x)g′(x)
  4. 除法法则
    (f(x)g(x))′=f′(x)g(x)−f(x)g′(x)g2(x)(g(x)≠0)\left( \frac{f(x)}{g(x)} \right)' = \frac{f'(x) g(x) - f(x) g'(x)}{g^2(x)} \quad (g(x) \neq 0)(g(x)f(x)​)′=g2(x)f′(x)g(x)−f(x)g′(x)​(g(x)=0)

四、复合函数的导数（链式法则）

若 y=f(u)y = f(u)y=f(u)，u=g(x)u = g(x)u=g(x)，则复合函数 y=f(g(x))y = f(g(x))y=f(g(x)) 的导数为：

  3. 两个三角函数的n阶导
