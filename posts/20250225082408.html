йҖҡиҝҮдёҖдёӘжһҒз®Җзҡ„ж—…иЎҢе•Ҷй—®йўҳпјҲTSPпјүжқҘиҜҰз»ҶиҜҙжҳҺиҡҒзҫӨз®—жі•зҡ„жӯҘйӘӨгҖӮеҒҮи®ҫжңү 3 дёӘеҹҺеёӮпјҲA, B, CпјүпјҢеҹҺеёӮй—ҙзҡ„и·қзҰ»еҰӮдёӢпјҡ

зӣ®ж ҮжҳҜжүҫеҲ°дёҖжқЎжңҖзҹӯи·Ҝеҫ„пјҢд»ҺжҹҗдёӘеҹҺеёӮеҮәеҸ‘пјҢи®ҝй—®жүҖжңүеҹҺеёӮеҗҺеӣһеҲ°иө·зӮ№гҖӮ

жӯҘйӘӨ 1пјҡеҲқе§ӢеҢ–еҸӮж•°

  • дҝЎжҒҜзҙ зҹ©йҳөпјҡжҜҸжқЎи·Ҝеҫ„еҲқе§ӢдҝЎжҒҜзҙ и®ҫдёә П„вӮҖ = 1гҖӮ
  • иҡӮиҡҒж•°йҮҸпјҡ2 еҸӘиҡӮиҡҒпјҲAnt1, Ant2пјүгҖӮ
  • жҢҘеҸ‘зҺҮ ПҒ = 0.3пјҲжҜҸж¬Ўиҝӯд»ЈеҗҺдҝЎжҒҜзҙ еҮҸе°‘30%пјүгҖӮ
  • еҗҜеҸ‘ејҸеӣ еӯҗ Оұ=1, ОІ=2пјҲдҝЎжҒҜзҙ е’Ңи·қзҰ»зҡ„жқғйҮҚпјүгҖӮ
  • дҝЎжҒҜзҙ еўһејәеёёж•° Q = 10гҖӮ

еҲқе§ӢдҝЎжҒҜзҙ иЎЁпјҡ

и·Ҝеҫ„ дҝЎжҒҜзҙ 
AвҶ’B 1
AвҶ’C 1
BвҶ’A 1
BвҶ’C 1
CвҶ’A 1
CвҶ’B 1

жӯҘйӘӨ 2пјҡиҡӮиҡҒжһ„е»әи·Ҝеҫ„

жҜҸеҸӘиҡӮиҡҒйҡҸжңәйҖүжӢ©дёҖдёӘиө·зӮ№пјҢйҖҗжӯҘи®ҝй—®жүҖжңүеҹҺеёӮгҖӮ

иҡӮиҡҒ1зҡ„и·Ҝеҫ„пјҲеҒҮи®ҫиө·зӮ№дёәAпјүпјҡ

  1. д»ҺAеҮәеҸ‘пјҢеҸҜйҖүBжҲ–CгҖӮ
  2. и®Ўз®—йҖүжӢ©жҰӮзҺҮпјҡ
    • и·Ҝеҫ„AвҶ’Bзҡ„еҗёеј•еҠӣпјҡ
      ( (П„_{AB})^Оұ \times (1/и·қзҰ»_{AB})^ОІ = 1^1 \times (1/2)^2 = 0.25 )
    • и·Ҝеҫ„AвҶ’Cзҡ„еҗёеј•еҠӣпјҡ
      ( 1^1 \times (1/3)^2 вүҲ 0.111 )
    • жҖ»еҗёеј•еҠӣпјҡ0.25 + 0.111 вүҲ 0.361
    • жҰӮзҺҮпјҡ
      • P(AвҶ’B) = 0.25 / 0.361 вүҲ 69.2%
      • P(AвҶ’C) = 0.111 / 0.361 вүҲ 30.8%
  3. жҢүжҰӮзҺҮйҖүжӢ©и·Ҝеҫ„пјҡеҒҮи®ҫиҡӮиҡҒ1йҖүжӢ© AвҶ’BгҖӮ
  4. еҲ°иҫҫBеҗҺпјҢеҸӘеү©еҹҺеёӮCжңӘи®ҝй—®пјҢеҝ…йЎ»йҖүжӢ© BвҶ’CгҖӮ
  5. жңҖеҗҺд»ҺCиҝ”еӣһиө·зӮ№AпјҢи·Ҝеҫ„дёә CвҶ’AгҖӮ

иҡӮиҡҒ1зҡ„е®Ңж•ҙи·Ҝеҫ„пјҡA вҶ’ B вҶ’ C вҶ’ A
жҖ»и·қзҰ»пјҡ2 (AвҶ’B) + 4 (BвҶ’C) + 3 (CвҶ’A) = 9 km

иҡӮиҡҒ2зҡ„и·Ҝеҫ„пјҲеҒҮи®ҫиө·зӮ№дёәBпјүпјҡ

  1. д»ҺBеҮәеҸ‘пјҢеҸҜйҖүAжҲ–CгҖӮ
  2. и®Ўз®—йҖүжӢ©жҰӮзҺҮпјҡ
    • и·Ҝеҫ„BвҶ’Aзҡ„еҗёеј•еҠӣпјҡ( 1^1 \times (1/2)^2 = 0.25 )
    • и·Ҝеҫ„BвҶ’Cзҡ„еҗёеј•еҠӣпјҡ( 1^1 \times (1/4)^2 = 0.0625 )
    • жҖ»еҗёеј•еҠӣпјҡ0.25 + 0.0625 = 0.3125
    • жҰӮзҺҮпјҡ
      • P(BвҶ’A) = 0.25 / 0.3125 = 80%
      • P(BвҶ’C) = 0.0625 / 0.3125 = 20%
  3. жҢүжҰӮзҺҮйҖүжӢ©и·Ҝеҫ„пјҡеҒҮи®ҫиҡӮиҡҒ2йҖүжӢ© BвҶ’AгҖӮ
  4. еҲ°иҫҫAеҗҺпјҢеҸӘеү©еҹҺеёӮCжңӘи®ҝй—®пјҢеҝ…йЎ»йҖүжӢ© AвҶ’CгҖӮ
  5. жңҖеҗҺд»ҺCиҝ”еӣһиө·зӮ№BпјҢи·Ҝеҫ„дёә CвҶ’BгҖӮ

иҡӮиҡҒ2зҡ„е®Ңж•ҙи·Ҝеҫ„пјҡB вҶ’ A вҶ’ C вҶ’ B
жҖ»и·қзҰ»пјҡ2 (BвҶ’A) + 3 (AвҶ’C) + 4 (CвҶ’B) = 9 km

жӯҘйӘӨ 3пјҡжӣҙж–°дҝЎжҒҜзҙ 

еұҖйғЁжӣҙж–°пјҲжҜҸеҸӘиҡӮиҡҒиҙЎзҢ®дҝЎжҒҜзҙ пјүпјҡ

  • дҝЎжҒҜзҙ еўһйҮҸе…¬ејҸпјҡ( О”П„ = Q / \text{и·Ҝеҫ„жҖ»й•ҝеәҰ} )
  • иҡӮиҡҒ1пјҲи·Ҝеҫ„й•ҝеәҰ9пјүпјҡжҜҸжқЎиҫ№еўһеҠ  ( О”П„ = 10/9 вүҲ 1.11 )
    • жӣҙж–°и·Ҝеҫ„пјҡAвҶ’B, BвҶ’C, CвҶ’A
  • иҡӮиҡҒ2пјҲи·Ҝеҫ„й•ҝеәҰ9пјүпјҡжҜҸжқЎиҫ№еўһеҠ  ( О”П„ = 10/9 вүҲ 1.11 )
    • жӣҙж–°и·Ҝеҫ„пјҡBвҶ’A, AвҶ’C, CвҶ’B

жҢҘеҸ‘еҗҺдҝЎжҒҜзҙ пјҡ

жҜҸжқЎи·Ҝеҫ„зҡ„дҝЎжҒҜзҙ е…ҲжҢҘеҸ‘30%пјҢеҶҚеўһеҠ иҡӮиҡҒиҙЎзҢ®пјҡ

  • еҺҹдҝЎжҒҜзҙ пјҡ1 вҶ’ жҢҘеҸ‘еҗҺпјҡ1 Г— (1-0.3) = 0.7
  • ж–°еўһдҝЎжҒҜзҙ пјҡ
    • AвҶ’B: 0.7 + 1.11 вүҲ 1.81
    • BвҶ’C: 0.7 + 1.11 вүҲ 1.81
    • CвҶ’A: 0.7 + 1.11 вүҲ 1.81
    • BвҶ’A: 0.7 + 1.11 вүҲ 1.81
    • AвҶ’C: 0.7 + 1.11 вүҲ 1.81
    • CвҶ’B: 0.7 + 1.11 вүҲ 1.81

жӯӨж—¶жүҖжңүи·Ҝеҫ„дҝЎжҒҜзҙ зӣёеҗҢпјҲеӣ дёәдёӨеҸӘиҡӮиҡҒи·Ҝеҫ„й•ҝеәҰзӣёеҗҢпјүпјҢдҪҶиӢҘжҹҗеҸӘиҡӮиҡҒжүҫеҲ°жӣҙзҹӯи·Ҝеҫ„пјҢе…¶иҙЎзҢ®зҡ„дҝЎжҒҜзҙ дјҡжӣҙй«ҳгҖӮ

жӯҘйӘӨ 4пјҡз¬¬дәҢж¬Ўиҝӯд»Ј

еҒҮи®ҫеңЁз¬¬дәҢж¬Ўиҝӯд»ЈдёӯпјҢиҡӮиҡҒ1йҖүжӢ©и·Ҝеҫ„ AвҶ’CвҶ’BвҶ’AпјҲжҖ»й•ҝеәҰ3+4+2=9 kmпјүпјҢиҡӮиҡҒ2йҖүжӢ© BвҶ’CвҶ’AвҶ’BпјҲжҖ»й•ҝеәҰ4+3+2=9 kmпјүпјҢдҝЎжҒҜзҙ жӣҙж–°дёҺз¬¬дёҖж¬Ўзұ»дјјгҖӮ

з»ҸиҝҮеӨҡж¬Ўиҝӯд»ЈеҗҺпјҢеҰӮжһңжҹҗжқЎи·Ҝеҫ„пјҲеҰӮAвҶ’BвҶ’CвҶ’Aпјүиў«йў‘з№ҒйҖүжӢ©дё”и·қзҰ»жӣҙзҹӯпјҢе…¶дҝЎжҒҜзҙ дјҡжҳҫи‘—й«ҳдәҺе…¶д»–и·Ҝеҫ„пјҢжңҖз»Ҳеј•еҜјиҡӮиҡҒж”¶ж•ӣеҲ°жңҖдјҳи§ЈгҖӮ

е…ій”®зӮ№жҖ»з»“

  1. жӯЈеҸҚйҰҲпјҡзҹӯи·Ҝеҫ„иў«жӣҙеӨҡиҡӮиҡҒйҖүжӢ© вҶ’ дҝЎжҒҜзҙ жө“еәҰжӣҙй«ҳ вҶ’ еҗҺз»ӯиҡӮиҡҒжӣҙеҖҫеҗ‘дәҺйҖүжӢ©е®ғгҖӮ
  2. жҢҘеҸ‘жңәеҲ¶пјҡйҒҝе…ҚдҝЎжҒҜзҙ ж— йҷҗз§ҜзҙҜпјҢи®©з®—жі•иғҪи·іеҮәеұҖйғЁжңҖдјҳгҖӮ
  3. еҗҜеҸ‘ејҸеӣ еӯҗпјҡОІ и¶ҠеӨ§пјҢиҡӮиҡҒи¶ҠеҖҫеҗ‘дәҺйҖүжӢ©и·қзҰ»зҹӯзҡ„и·Ҝеҫ„гҖӮ

жңҖз»Ҳж•Ҳжһң

  • еҰӮжһңеӯҳеңЁжӣҙзҹӯи·Ҝеҫ„пјҲдҫӢеҰӮAвҶ’BвҶ’CвҶ’AжҖ»й•ҝ9 kmпјҢиҖҢAвҶ’CвҶ’BвҶ’AжҖ»й•ҝ7 kmпјүпјҢз®—жі•дјҡйҖҡиҝҮдҝЎжҒҜзҙ е·®ејӮйҖҗжёҗж”¶ж•ӣеҲ°жӣҙдјҳи§ЈгҖӮ
  • з»ҸиҝҮеӨҡж¬Ўиҝӯд»ЈпјҢдҝЎжҒҜзҙ зҹ©йҳөдјҡжё…жҷ°еҸҚжҳ жңҖдјҳи·Ҝеҫ„гҖӮ
